# PROSJEKT: MINI SOC

<h3>Mini Security Operations Center er en webapplikasjon som er en simulasjon/rollespill av et ordentlig SOC system der min PC er main "server". </h3>

---

<h3>Der du kan se logging av innlogginger, alerts, PC/"Server" info (PSUTIL) og en liten oppsummerings statistikk av hvor mange logs av hver "event_type".<h3>
